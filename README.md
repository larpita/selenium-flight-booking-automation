# Selenium Flight Booking Automation

Automated end-to-end flight booking workflow on a demo travel website.

## Features Automated
- Select source and destination cities
- Search available flights
- Choose flight and complete booking flow

## Tech Stack
- Python
- Selenium WebDriver

## How to Run
1. Install requirements
2. Run test_flight_booking.py

## Learning Outcome
- Hands-on Selenium automation
- Locator strategies and waits
